Добро пожаловать в ScanDoc! 

Внутри этой папки лежит 2 установщика, один содержит только CLI компонент, другой только GUI

Руководство по установке ScanDoc (только Windows)

Перед установкой отключите антивирус!

Необходимое ПО:
1. Python 3.12
2. Tesseract 5.5.0
3. GhostScript (Для обработки .pdf)
4. Кодировка UTF-8 в консоли
5. WebView2 (обязательный компонент для GUI)

(Первые 3 элемента можно установить прямо из установщика)

Кодировка UTF-8 в консоли нужна для корректной работы Tesseract, ее можно установить следуя следующему алгоритму:
1) Win + R
2) Введите regedit
3) Перейдите в каталог: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Command Processor
4) Найдите строковый параметр Autorun (тип REG_SZ), если его нет, создайте новый
5) Установите значение @chcp 65001>nul
6) Нажмите Ок

Tesseract 
Обязательно во время установки обязательно выберите `Additional language data`
- Russian
- English

WebView2 - можно установить с официального сайта Microsoft (под вашу систему)

НАСТОЯТЕЛЬНО НЕ РЕКОМЕНДУЕТСЯ УСТАНАВЛИВАТЬ ScanDoc В Program Files/Program Files (x86) и подобные системные папки (не касается tesseract, python, ghostScript)

Обязательно добавьте в PATH: tesseract, python, ghostScript
*Проверьте что python не от Microsoft/WindowsApps*

Для работы с приложением можете воспользоваться руководством по использованию ScanDoc

* Примечание *
Для получения доступа к обоим компонентам сразу:
1) Скачайте архив и разорхивируйте с https://github.com/Fewerim/ScanDoc
2) Внутрь папки lib/service скачайте и разорхивируйте архив с https://github.com/hahahhha/doc-recognition-new внутрь папки scanPy
3) В итоговом результате внутри lib/service/scanPy должно лежать содержимое репозитория из пункта 2
